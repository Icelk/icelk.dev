httPWM

httPWM is a web application and website combo which enables controlling of PWM devices on a Raspberry Pi.

Here is the electrical circuit I'm using to connect the software with reality.

More info can be found on GitHub.

The front-end currently supports one transition per day (wake-up clock) and schedulers to execute transitions every day, once per week, or at a arbitrary date and time. It also supports several interpolation modes with a extensible API for future improvements.
