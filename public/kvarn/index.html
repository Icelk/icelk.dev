Kvarn

Kvarn is a next-generation web server designed for performance and flexibility.

Check out the Github page at Icelk/kvarn for more info.

It features a pluggable extension system ontop of which many great features are built on, both internally and through Kvarn's extension library.

Right now, possibly as you read this, I'm improving the code-base and adding new features in the main branch. See the v0.2 branch for a "stable" version.
